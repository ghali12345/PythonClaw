## 🧩 What is PythonClaw?

PythonClaw is an AI agent designed to work on your Windows computer. It can remember things, learn from new information, and talk to you using your voice. It also has a web dashboard to control settings and supports many messaging channels like Telegram. All this comes in a user-friendly package built purely with Python.

You don’t need any programming experience to use it. This guide will help you get PythonClaw running on your Windows PC in simple steps.

## 💻 System Requirements

Before starting, make sure your computer meets these basic needs:

- Operating System: Windows 10 or higher
- Processor: Intel i3 or equivalent
- Memory: 4 GB RAM minimum
- Storage: 1 GB free disk space
- Network: Active internet connection for setup and online features

PythonClaw runs on most modern Windows machines. If your computer is older, performance may vary.

## 🔧 Using the Web Dashboard

The dashboard lets you manage PythonClaw’s settings and features without typing commands.

- The dashboard runs locally on your PC, no internet needed to operate basic features.
- Open your browser and navigate to `http://localhost:5000` if it doesn’t open automatically.
- From there, you can add memory notes, check conversations, enable voice input, and view logs.
- You can also set which chat services to connect, including Telegram.

## 🔄 Running PythonClaw as a Background Service (Daemon)

If you want PythonClaw to stay active without having the dashboard open:

- The software includes a daemon mode.
- Enable this in the dashboard settings under "Service Mode".
- When enabled, PythonClaw runs silently in the background.
- This keeps your AI agent ready for commands from connected apps or messaging platforms.
